# ts_fullstack_app

Employee fullstack app is a dashboard which allows to create, read, update and delete information about employees. Also includes user registration and authorization.

Stack:

Backend:
- Express JS
- Prisma Studio ORM
- SQLite
- JWT

Frontend:
- TypeScript
- React
- Redux
- Redux Toolkit/ RTK Query
- React Router DOM
- Ant Design
- Vite
